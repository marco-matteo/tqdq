# TQDT (LB2 - Applikation)
## Start
Zum Starten über Docker können die folgenden Befehle ausgeführt werden:
```bash
cd doqqer
docker compose up
```
Danach kann auf [`localhost:8080`](http://localhost:8080) auf die App (WAF) zugegriffen werden
